# 🎮 Rock Paper Scissors Game

A simple and interactive web-based game where users can play Rock, Paper, Scissors against the computer.

---

## 🚀 Features
- Interactive UI with buttons for user input  
- Random computer move generation  
- Real-time result display (Win/Lose/Draw)  
- Clean and responsive design  
- Beginner-friendly implementation  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

---

## ⚙️ How It Works
- User selects Rock, Paper, or Scissors  
- Computer randomly chooses an option  
- Game logic compares both choices  
- Result is displayed instantly  

---

## ▶️ How to Run
1. Download or clone the repository  
2. Open `index.html` in any browser  
3. Start playing 🎮
